Fork the repository

The source code of this application is already hosted on GitHub, so you just need to go to canihavesomecoffee/ci_workshop and then fork the repository.

This forking principle is used a lot within the open source community, because you can play around on your own personal copy and do some development work without the need to bother others (or if you disagree with the developers, you just go your own way ;) ).
